⚡ QUICK SETUP (5 MINUTES)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

FOR EACH SUBSCRIPTION (premium_monthly & premium_yearly):

1. CREATE SUBSCRIPTION
   ├─ Click "Create subscription"
   ├─ Product ID: premium_monthly (or premium_yearly)
   ├─ Name: Premium Monthly (or Premium Yearly)
   └─ Click "Create"

2. ADD BASE PLAN
   ├─ Click "Add base plan"
   ├─ Base plan ID: monthly-autorenewing (or yearly-autorenewing)
   ├─ Type: Auto-renewing ✓
   ├─ Billing period: 1 Month (or 1 Year)
   └─ Click "Continue"

3. SET PRICING
   ├─ Click "Set prices"
   ├─ Enter price: $2.99 (monthly) or $19.99 (yearly)
   └─ Click "Apply prices"

4. ACTIVATE
   └─ Click "Activate" ✓

🎁 OPTIONAL: FREE TRIAL
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

After creating base plan:
   ├─ Click "Add offer"
   ├─ Offer ID: free-trial-7days
   ├─ Eligibility: New customer acquisition ✓
   ├─ Add Phase 1: Free trial for 7 days
   ├─ Add Phase 2: Regular price (automatic)
   └─ Click "Save" → "Activate"

🧪 TESTING SETUP
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Before testing:
   1. Go to: Setup → License testing
   2. Add your email address (comma-separated for multiple)
   3. Upload APK to Internal Testing track
   4. Wait 1-2 hours for products to propagate
   5. Install app on device with test account
   6. Navigate to Settings → Mobile Subscription
   7. Products should appear! ✓

⚠️  COMMON ISSUES
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

❌ "No subscriptions found"
   → Product ID typo (must be EXACT)
   → Base plan not activated
   → App not uploaded to Internal Testing
   → Wait 1-2 hours for propagation

❌ Products show "pending"
   → Wait 1-2 hours after creating/activating

❌ Can't find "Subscriptions" menu
   → Need to upload APK first
   → Need Google Merchant Account setup
