Option 1: Use Capacitor Camera Plugin (supports video!)

The official Capacitor Camera plugin can record video, but only if you set the correct options.

Install:
npm install @capacitor/camera
npx cap sync

Use:
import { Camera, CameraSource, CameraResultType } from '@capacitor/camera';

const recordVideo = async () => {
  const video = await Camera.getPhoto({
    source: CameraSource.Camera,
    resultType: CameraResultType.Uri,
    allowEditing: false,
    saveToGallery: true,
    quality: 90,
    presentationStyle: 'fullscreen',
    // IMPORTANT:
    captureMode: "VIDEO"  // this enables video recording
  });

  console.log(video);
};

âœ… Option 2: Use @capacitor-community/media-capture

This plugin gives full control over recording video + audio.

Install:
npm install @capacitor-community/media-capture
npx cap sync

Example:
import { MediaCapture } from '@capacitor-community/media-capture';

const recordVideo = async () => {
  const result = await MediaCapture.captureVideo({
    limit: 1,
    duration: 15
  });

  console.log(result);
};
